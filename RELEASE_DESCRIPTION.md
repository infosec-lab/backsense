## 🛠️ **Installation**

1. **Download** `BackSense-v2.0.0.jar`
2. **Open** Burp Suite Professional
3. **Navigate** to Extensions → Extensions
4. **Click** "Add" → "Extension Type: Java"
5. **Select** the downloaded JAR
6. **Click** "Next" and "Close"
